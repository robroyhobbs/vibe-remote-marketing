## View locally

```
open index.html
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Your page will be live at `https://<you>.github.io/vibe-remote-marketing`
