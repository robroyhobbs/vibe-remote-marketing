Launch goal
发布目标

Don’t chase awareness. Chase conversations and installs.

不要追求曝光，要追求对话和安装。

Your first launch is not about scale. It is about compressing learning into one intense week.

第一次发布不是为了规模，而是把学习速度压缩到一周之内。

Success target

成功目标

100 downloads, 20+ meaningful conversations, 10 paired users, 3 repeat users, 1 clear candidate for PRO.

100 次下载、20+ 次有价值对话、10 个成功配对用户、3 个复访用户、1 个明确的 PRO 候选人。

Primary asset

核心资产

One 30–60 second demo video. If you only make one thing, make this.

一个 30–60 秒 demo 视频。如果只做一个资产，就做这个。

Primary CTA

主 CTA

Install the app, pair your Mac, send your first prompt.

安装 App，配对 Mac，发送第一条提示词。

Launch asset checklist
发布资产清单

The minimum proof package

最小可用证据包

Must-have assets

必须准备

  • Demo video: QR scan → connected → prompt → output → wow.
  • 3 clean screenshots: dashboard, session output, localhost preview.
  • 1 sentence description: “Control Claude Code from your iPhone, no VPN needed.”
  • 1 founder story: why you built it instead of using SSH + Tailscale.
  • 1 copyable install flow: app + agent + first prompt.
  • Demo 视频：扫码 → 连接 → 提示词 → 输出 → wow。
  • 3 张清晰截图：dashboard、session 输出、localhost 预览。
  • 一句话介绍：“在 iPhone 上控制 Claude Code，无需 VPN。”
  • 1 段创始人故事：为什么你没有继续用 SSH + Tailscale。
  • 可复制的安装流程：App + agent + 第一条提示词。

Nice-to-have assets

加分项

  • Before/after comparison against SSH + tmux on mobile
  • “Built 100% by AI” behind-the-scenes screenshots
  • Short testimonial from a trusted developer friend
  • App Store screenshots localized in English + Chinese
  • A “first 3 prompts” onboarding page linked from the site
  • 与 SSH + tmux 手机方案的前后对比
  • “100% 由 AI 构建”的幕后截图
  • 来自可信开发者朋友的短评
  • 英文 + 中文 App Store 截图
  • 站点内链接的“前 3 条提示词”上手页
